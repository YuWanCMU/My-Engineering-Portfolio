GPU-based Interval Arithmetic Optimiser

Solving large-scale highly non-convex non-linear problems with rigorous guarantees on the optimum.

Introduction

This project develops a GPU-accelerated interval arithmetic optimiser for highly non-convex, nonlinear engineering problems. The aim is to obtain mathematically rigorous bounds on the true optimum rather than approximate local solutions. This is valuable for safety-critical design tasks, where under-estimating stresses or failure probabilities can lead to costly or dangerous outcomes.

Methods

Classical objective functions from structural and mechanical design were reformulated using interval arithmetic so that rounding errors and model uncertainty are enclosed. A CUDA/Numba implementation evaluates millions of interval sub-regions in parallel on the GPU. The algorithm iteratively prunes regions that cannot contain the global optimum and subdivides promising regions to refine the solution bounds. Nsight profiling was used to tune memory access patterns and thread/block configurations.

Results

The optimiser successfully solved benchmark non-convex problems with guaranteed enclosures on the global optimum. Compared with a CPU reference implementation, the GPU version enabled much finer partitioning of the design space within practical runtimes. The final outputs are heat-map plots of surviving regions and corresponding rigorous bounds on the objective.

Discussion

This work demonstrates that interval arithmetic, combined with GPU parallelism, can deliver both rigour and scalability for engineering optimisation. The approach is particularly useful when design decisions must be justified with guaranteed bounds instead of single approximate values.

My Role

I designed the optimisation algorithm, implemented all CUDA/Numba kernels, carried out performance profiling, and prepared the visualisations of surviving regions and solution bounds.
