Microneedle Layout Optimisation

Optimising a microneedle array layout for maximum flow while respecting geometric and manufacturing constraints.

Introduction

This project focuses on optimising the layout of hollow microneedles that transport blood or drugs into a microfluidic chip. The goal is to maximise flow efficiency while satisfying strict geometric and manufacturing constraints. The work supports the development of less painful and more reliable medical sampling devices.

Methods

A parametric CAD model of the microneedle array and connecting microchannels was created. The flow behaviour was modelled using Hagen–Poiseuille theory and simplified microfluidic equations. SciPy's constrained trust-region optimisation method (trust-constr) was used to tune key design variables such as channel diameters, spacing, and manifold geometry. Constraints ensured manufacturability, avoidance of channel blockage, and compatibility with the target microfluidic chip.

Results

The optimiser delivered a layout that satisfied all geometric and manufacturing constraints while achieving a higher predicted max-flow capacity than the baseline design. Visualisations include 3D CAD renderings of the optimised layout and labelled diagrams showing flow paths from the inlet port to the microfluidic chip.

Discussion

By combining physics-based modelling with numerical optimisation, this project demonstrates a practical workflow for improving micro-scale medical devices without extensive physical prototyping. The methodology can be transferred to other microfluidic or biomedical layouts where flow performance and geometric constraints must be balanced.

My Role

I built the parametric CAD model, derived the simplified flow model, implemented the optimisation using SciPy, and generated the final visualisations and explanatory diagrams for the optimised design.
